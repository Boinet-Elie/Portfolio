# Portfolio Elie Boinet

Bienvenue sur mon portfolio en ligne ! Ce projet présente mes compétences, mes projets et mes informations de contact. Il a été conçu pour montrer mon travail de manière professionnelle et interactive.

## 🚀 Aperçu du projet

Ce portfolio est un site web statique développé avec **HTML**, **CSS** et **JavaScript**. Il comprend plusieurs sections :
- **À propos** : Une présentation de moi-même et de mes compétences.
- **Compétences** : Une grille Bento interactive pour mettre en avant mes compétences techniques.
- **Projets** : Un carrousel pour présenter mes projets avec des images, des descriptions et des liens.
- **Contact** : Un formulaire de contact fonctionnel pour me contacter directement.

## 🌟 Fonctionnalités

- **Design moderne et responsive** : Le site s'adapte à tous les écrans (ordinateur, tablette, mobile).
- **Animations fluides** : Effets de survol, transitions et animations CSS pour une expérience utilisateur agréable.
- **Carrousel interactif** : Les projets sont présentés dans un carrousel avec des boutons de navigation.
- **Formulaire de contact** : Permet aux visiteurs de m'envoyer un message directement depuis le site.

## 🛠️ Technologies utilisées

- **HTML** : Structure du site.
- **CSS** : Styles et animations.
- **JavaScript** : Interactivité (carrousel, effets dynamiques).
- **Font Awesome** : Icônes pour les réseaux sociaux et les compétences.
- **Google Fonts** : Police Poppins pour une typographie moderne.
