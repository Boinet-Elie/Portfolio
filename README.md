# Portfolio — Elie Boinet

Site vitrine personnel développé dans le cadre du BTS SIO SLAM. Présente mes compétences, projets, documents professionnels et veille technologique.

## Pages

| Page | Description |
|------|-------------|
| `Index.html` | Accueil — hero, à propos, compétences, présentation BTS SIO |
| `Project.html` | Projets — grille de cartes avec liens vers les dépôts GitHub |
| `Tableau_Synthese.html` | Tableau de synthèse BTS (image + téléchargement PDF) |
| `Veille.html` | Veille technologique (IA, hardware, éco-numérique) |
| `Contact.html` | Formulaire de contact (Formspree) |

## Technologies

- **HTML / CSS / JS** — vanilla, aucun framework
- **Font Awesome 6** — icônes
- **Google Fonts** — Inter + JetBrains Mono
- **Formspree** — soumission du formulaire de contact

## Design

- Palette monochrome noir/blanc avec effets glassmorphism
- Esthétique terminal/code pour les titres et éléments décoratifs
- Responsive (mobile, tablette, desktop)
- Animations : typewriter, fade-in au scroll, carousel auto-défilant
